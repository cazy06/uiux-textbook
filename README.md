# UIUXデザインの教科書
### ギャレットの5レイヤーモデルで学ぶ体系的デザイン
**UI/UX Design Textbook — A Systematic Guide through Garrett's 5-Layer Model**

[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

## 概要 / Overview

Jesse James Garrett の **Elements of User Experience（5レイヤーモデル）** を軸に、UX・UIデザインの概念・手法・隣接領域を体系的に整理した日英バイリンガルの教科書草稿です。

A bilingual (Japanese/English) textbook draft systematically covering UX/UI design concepts, methods, and adjacent fields, structured around Jesse James Garrett's **Elements of User Experience (5-Layer Model)**.

### 収録内容 / Contents

| # | 章 / Chapter |
|---|---|
| 序章 | UIとUXの定義・歴史・用語の起源 |
| Part 1 | 戦略層 — ユーザーリサーチ・ペルソナ・JTBD |
| Part 2 | スコープ層 — 要件定義・コンテンツ戦略 |
| Part 3 | 構造層 — IA・インタラクションデザイン |
| Part 4 | 骨格層 — ワイヤーフレーム・UIパターン |
| Part 5 | 表層 — ビジュアルデザイン・モーション |
| Part 6 | ユーザビリティ評価・Nielsenの10原則・Shneidermanの8原則 |
| 付録 A–J | ブランディング・コンテンツ・サービスデザイン・経営戦略・UX成熟度・フロントエンド開発・未来指向デザイン・ゲームデザイン・マーケティング戦略・イノベーション研究 |

### 特徴 / Features

- 🌐 **日英バイリンガル** — ボタン1つで日本語/英語を切り替え
- 📱 **レスポンシブ** — デスクトップ・タブレット・モバイル対応
- 🔗 **単一HTMLファイル** — `index.html` 1ファイルで完結
- 📖 **OSSスタイル** — Issues・Pull Requestで誰でも改善提案が可能
